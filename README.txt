# 🛒 Shopping List App

Este é um projeto de aplicação de lista de compras desenvolvido em **React Native**, como parte do curso da Rocketseat. O objetivo é praticar conceitos fundamentais da criação de interfaces móveis, gerenciamento de estado, navegação e persistência de dados.

## 🚀 Funcionalidades

- Adicionar itens à lista de compras
- Remover itens da lista
- Marcar itens como comprados
- Salvar dados localmente no dispositivo
- Interface amigável e responsiva

## 🧱 Tecnologias Utilizadas

- [React Native](https://reactnative.dev/)
- [Expo](https://expo.dev/) (ambiente de desenvolvimento)
- [AsyncStorage](https://react-native-async-storage.github.io/async-storage/) (persistência local)

## 📱 Layout

O layout segue os padrões modernos de usabilidade e foi inspirado nas diretrizes ensinadas no curso. O design foca em simplicidade, legibilidade e facilidade de interação.

## 🛠️ Como Executar

1. Clone o repositório:

   ```bash
   git clone https://github.com/Divalter/comprar.git

2. Instale as dependências:

    ```bash
    cd nome-do-repositorio
    npm install

3. Inicie a aplicação:

    ```bash
    npx expo start

4. Escaneie o QR code com o aplicativo Expo Go para rodar no dispositivo físico ou utilize um emulador.
